# django-orm-watching-storage

Учебный проект "Пульт охраны банка", написанный в рамках курса "Знакомство с Django: ORM" [dvmn.org](dvmn.org). Реализован вывод информации из базы данных банка:
- об активных пропусках (имя владельцаб код пропуска, дата регистрации);
- о всех посещениях хранилища для каждого активного пропуска (дата посещения, продолжительность визита, был ли визит продолжительным);
- о сотрудниках, находящихся в хранилище сейчас (имя владельцаб код пропуска, дата регистрации).


### Цель проекта

Код написан в образовательных целях на онлайн-курсе для веб-разработчиков [dvmn.org](https://dvmn.org/).
